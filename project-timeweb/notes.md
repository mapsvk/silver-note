##  Корпоративная цветовая палитра

- Светло-серый / Холодный серый ( #F5F7FA ) — Внешний мир (Интернет). Используется как подложка для зоны за пределами сервера. На этой линии мы размещен домен ****.ru и входящий трафик.
- Пастельно-синий ( #E1F5FE ) — Контур управления DevOps. Цвет надежности и системности. В этой зоне ноутбуки и ПК. Линию SSH-доступа к порту 22 ( #0288D1 ). 
- Пастельно-оранжевый / Песочный ( #FFF3E0 ) — Клиентский контур (Пользователи). Обозначает зону с полудоверенным трафиком, который требует жесткой фильтрации. Стрелочки ( #E65100 )
- Глубокий темно-синий / Грифельный ( #1E293B ) — Граница облачного сервера. Сам прямоугольник сервера Timeweb с IP 1.1.1.1 должен иметь четкую, строгую темную границу.
- Пастельно-зеленый ( #E8F5E9 ) — Внутренний доверенный контур (Docker). Зеленый цвет означает «безопасно, трафик проверен и изолирован». Здесь будут находиться базы данных и внутренние сети контейнеров.

P.S.
Возожно это не корпоративные цвета и это совсем другая цветовая схема (поразмышлять.)
