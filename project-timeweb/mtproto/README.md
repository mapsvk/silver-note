## Подготовка секретного ключа

Сгенерируйте надежный ключ для Fake TLS (с префиксом ee) одной командой в терминале:bash

echo "$(openssl rand -hex 16)"


## Открыть порт на сервере (UFW)

sudo ufw allow 443/tcp


## Привязка промо-канала (канал-спонсор)

Если вы хотите, чтобы у всех пользователей вашего прокси вверху списка чатов отображался ваш (или чужой) Telegram-канал, его нужно зарегистрировать.

- В Telegram найдите официального бота @MTProxybot. 
- Отправьте команду /newproxy и следуйте инструкциям: 
- - Отправьте IP и порт вашего сервера (195.133.66.236:443).
- - Отправьте ваш секрет без префикса ee (7ce4c6173cd234ee143cf04793f47c52).
- Бот выдаст вам специальный шестнадцатеричный тег. Скопируйте его (он выглядит как набор букв и цифр, например 92b0c3...).
- Нажмите /setchannel, выберите ваш прокси и перешлите боту канал, который хотите продвигать.Обновите ваш файл compose.yaml, раскомментировав строчку TAG: yaml

```bash
environment:
  - SECRET=7ce4c6173cd234ee143cf04793f47c52
  - TLS_DOMAIN=google.com
  - TAG=скопированный_тег_от_бота
```

- Перезапустите контейнер: docker compose up -d. Канал появится у пользователей в течение часа.
